## Installation

First command: doctrine:migrations:load

mobus:fixtures:test - this command will load all app data.

mobus:tickets:command - this command will load all available tickets.

mobus:mobile:cache - this command will load all mobile lines to cache.

mobus:cache:test - this command will load all web lines to cache.

check:ticket:status - command for check cache version.
